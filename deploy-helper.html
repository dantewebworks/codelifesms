üêô GitHub + Vercel Deployment

üìÅ Files You Need:

  • index.js - Main webhook handler
  • package.json - Project configuration
  • vercel.json - Vercel settings

All files are already created for you!

Step 1: Create GitHub Repository

  1. Go to github.com and sign in
  2. Click "New repository" (green button)
  3. Repository name: telnyx-webhook
  4. Make it Public
  5. Click "Create repository"

Step 2: Upload Files to GitHub

  1. Click "uploading an existing file" link
  2. Drag and drop the 3 files: index.js, package.json, vercel.json
  3. Scroll down and click "Commit changes"

Step 3: Deploy to Vercel

  1. Go to vercel.com and sign in
  2. Click "New Project"
  3. Find your telnyx-webhook repository
  4. Click "Import"
  5. Click "Deploy" (don't change any settings)
  6. Wait 2 minutes for deployment to complete

Step 4: Get Your Webhook URL

After deployment, you'll see:

Your webhook URL: https://telnyx-webhook-[random].vercel.app/webhook

Copy this URL - you'll use it in your Telnyx settings!

üìÅ Direct Upload to Vercel

üìÅ Files You Need:

  • index.js - Main webhook handler
  • package.json - Project configuration
  • vercel.json - Vercel settings

Create a folder called "telnyx-webhook" and put these 3 files in it

Step 1: Prepare Files

  1. Create a new folder on your desktop called telnyx-webhook
  2. Copy the 3 files into this folder
  3. Make sure all files are directly in the folder (not in subfolders)

Step 2: Deploy to Vercel

  1. Go to vercel.com and sign in
  2. Click "New Project"
  3. Drag your telnyx-webhook folder onto the page
  4. Click "Deploy"
  5. Wait 2 minutes for deployment

Step 3: Get Your Webhook URL

After deployment completes:

Your webhook URL: https://your-project-name.vercel.app/webhook

Test it by visiting the URL in your browser!
